# 工资计算器 / Salary Calculator

多国工资计算工具，支持中国（个税+五险一金）与日本（年收、所得税、住民税简化）。内置中/英/日多语言。

- 国家切换：中国 / 日本
- 多语言：中文 / English / 日本語
- 模式（中国）：单月估算、累计预扣（按年度速算扣除表）
- 月度明细（中国）：工资、社保、公积金、专项附加可逐月调整，一键填充
- 日本模式：健康/年金封顶估算，含复兴特别税；住民税简化为10%+5000（第二年起）

## 运行
```bash
flutter pub get
flutter run -d web-server --web-port 8000   # Web 调试
# 或真机/模拟器：
flutter run
```

## 打包
- Web：`flutter build web`
- Android APK：`flutter build apk --release`（输出：`build/app/outputs/flutter-apk/app-release.apk`）

## 计算逻辑（摘要）
- 中国单月：按月度档位（3k/12k/...）+ 速算扣除。
- 中国累计：累计收入、五险一金、专项附加、基础扣除（5000×月），按年度档位（36k/144k/...）+ 速算扣除得累计税额，当月税 = 本期累计税 - 上期累计税，税额四舍五入到分。
- 日本：年收 -> 给与所得控除 -> 社保（健康/年金封顶、雇佣、40+护理）-> 基础/配偶/扶养扣除 -> 所得税表+2.1%复兴特别税；住民税简化为课税所得×10%+5000（首年可为0）。
