# 日本工资个税与社保计算要点（令和6-7，2024-2025）

> 日本个税年度：1/1 - 12/31。以下为常用简化规则，便于实现 JapanTaxEngine。

## 1. 社会保险（员工部分，便于估算）
- 计算基于标准报酬月额，可用费率估算并加封顶：
  - 健康保险：约 5.0%（东京协会健保 9.98% 的一半），月薪超 1,390,000 日元封顶。
  - 厚生年金：9.15%，月基数封顶 650,000 日元。
  - 雇佣保险：0.6%，无封顶。
  - 护理保险：约 0.9%，仅 40 岁及以上。

## 2. 给与所得控除（工资所得扣除，按年收入 A）
| 年收入区间 (日元) | 公式 |
| --- | --- |
| 0 ~ 1,625,000 | 550,000 |
| 1,625,001 ~ 1,800,000 | A × 0.4 - 100,000 |
| 1,800,001 ~ 3,600,000 | A × 0.3 + 80,000 |
| 3,600,001 ~ 6,600,000 | A × 0.2 + 440,000 |
| 6,600,001 ~ 8,500,000 | A × 0.1 + 1,100,000 |
| 8,500,001 以上 | 1,950,000（上限） |

> 最低档如计算值低于 550,000，仍按 550,000。

## 3. 个人扣除
- 基础扣除：480,000（合计所得 < 2,400 万通常适用）。
- 配偶扣除：380,000（配偶年收 < 1,030,000 且纳税人年收 < 9,000,000）。
- 扶养扣除（常用值）：
  - 一般扶养（16+）：380,000/人
  - 特定扶养（19-22）：630,000/人
  - 老人扶养（70+）：同居 580,000 / 非同居 480,000

## 4. 所得税税率表（源泉徴収/年调通用，课税所得 B）
| 区间 (日元) | 税率 | 速算扣除 |
| --- | --- | --- |
| 0 ~ 1,949,000 | 5% | 0 |
| 1,950,000 ~ 3,299,000 | 10% | 97,500 |
| 3,300,000 ~ 6,949,000 | 20% | 427,500 |
| 6,950,000 ~ 8,999,000 | 23% | 636,000 |
| 9,000,000 ~ 17,999,000 | 33% | 1,536,000 |
| 18,000,000 ~ 39,999,000 | 40% | 2,796,000 |
| 40,000,000 以上 | 45% | 4,796,000 |

- 复兴特别所得税：税额 × 1.021。
- 课税所得 = 年收入 - 给与所得控除 - 社保总额 - 基础扣除 - 扶养/配偶扣除。

## 5. 住民税（简化）
- 税率：10%（道府县 4% + 市町村 6%），均等割约 5,000/年。
- 首年通常不预扣，假设“第 2 年及以后”则：住民税 = 课税所得 × 10% + 5,000。首年可视为 0。

## 6. 开发实现要点（JapanTaxEngine）
输入：
- 年收入（Gross）
- 年龄（判定护理保险）
- 是否首年（决定是否计入住民税）
- 配偶有无
- 扶养人数（常规 38 万/人，简化）

计算流程：
1) 社保 = 健康 + 年金 + 雇佣 +（如 40+ 则加护理）；健康/年金按月基数封顶后 ×12。
2) 给与所得控除：按年收入查表。
3) 课税所得 = Gross - 社保 - 给与所得控除 - 480,000 - 扶养/配偶扣除（简化）。
4) 所得税 = (课税所得 × 税率 - 速算扣除) × 1.021。
5) 住民税 = 0（首年）或 课税所得 × 0.10 + 5,000（简化）。
6) 手取り = Gross - 社保 - 所得税 - 住民税。

> 定额减税（2024 年每人 4 万，所得税 3 万 + 住民税 1 万）暂不默认启用，可视需要加开关。
